# Cartella Immagini per GitHub / GitHub Images Folder

In questa cartella `public/` vanno collocate le immagini del sito web per essere caricate su GitHub ed essere servite correttamente dall'applicazione:

1. **`logo.png`**
   - Utilizzato nell'Header (barra di navigazione) e nel Footer.
   - Consigliato formato PNG con sfondo trasparente.

2. **`profilo.png`**
   - Utilizzato nella sezione d'introduzione (Hero) per la foto principale di Francesca.
   - Sfondo a contrasto o trasparente.

3. **`profilo2.png`**
   - Utilizzato nella sezione "Chi sono" per la seconda foto di Francesca (es. in studio).

4. **`logo-socialee.png`**
   - Logo dell'agenzia partner Socialee nel footer.

---

### Note di Sviluppo:
Nel codice, per requisiti di portabilità ed esportazione, tutti i percorsi delle immagini sono scritti in formato relativo senza la barra iniziale `/` (es. `logo.png`, `profilo.png`, `profilo2.png`, `logo-socialee.png`).
